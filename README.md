# ping-pong

Create file ```Credentials.php```:

```
<?php

$oldCredentials = array(
        'host' => 'db1.host.ch',
        'username' => 'me',
        'password' => 'me',
        'database' => 'me-temp',
        'port' => '3306',
);

$newCredentials = array(
        'host' => 'db1.host.ch',
        'username' => 'me',
        'password' => 'me',
        'database' => 'me',
        'port' => '3306',
);
```
